Framework used

  1. Source of truth = client's "All content must remain" document. Every edit cites the specific rule from the doc it enforces. No personal judgment; every change has a paper trail.
  2. Regulatory overlay: HSA Singapore Health Supplement Guidelines · US FDA DSHEA structure/function claim rules · EU food-supplement framework · Google Ads Health policy · YMYL content standards.
  3. Four hard constraints. (a) Educational, reference-style, neutral tone only — no promotional voice, no sales, no superiority over competitors, no product pitch. (b) Zero invention — if a claim cannot be verified against a real primary source, it is deleted, not softened. (c) No therapeutic / disease / repair / cure / anti-aging / guaranteed-outcome language; no disease names; no invented "therapy" terms. (d) No overstated scientific certainty — cautious, evidence-aware phrasing only.
  4. All external citations removed. Fake citations are a bigger compliance and SEO risk than no citations. The Reputifly template only auto-generates the Sources & Citations box if external <a> tags exist — once removed, the fabricated-source surface is eliminated entirely.
  5. Keyword discipline: stay within deer placenta, deer placenta supplement, health supplements. No cross-post cannibalization.
  6. Deterministic regex gate. Every rewritten post was scanned against a banned-phrase regex list before it was written live. Zero hits on all five.
  7. Titles preserved this pass. Per client direction, the existing post titles were not modified. Any title-level issues are noted in the per-post audit pages and left for a future pass.

Method

Each post was pulled live from the Reputifly agent API (GET /posts/<id>), passed to a dedicated Opus-class audit agent, rewritten against the framework above, scanned with a deterministic banned-phrase regex gate, and only then published back via PUT /posts/<id>. Post slugs, publish dates, categories, and tags were preserved unchanged. Every PUT returns a rollback-capable change ID in the Reputifly changelog — every edit in this report is fully reversible.

What this pass deliberately did not do

  • Did not invent any citation, study, DOI, researcher, or institution.
  • Did not replace removed external links with other external links — they stay removed.
  • Did not soften fabricated claims; fabrications were deleted.
  • Did not change post slugs, publish dates, or post IDs.
  • Did not alter categories or tags automatically.
  • Did not modify titles. Any title-level compliance issue is flagged in the per-post page for a future pass.
