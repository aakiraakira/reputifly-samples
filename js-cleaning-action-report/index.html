1 Issues Found

Critical problems identified during the Google Ads audit and landing page review.

4 Critical Issues Identified

Triple-Counting Conversions

3 conversion actions all set to Primary and included in account-level goals. Every single lead was being counted 3 times, corrupting the Maximize Conversions bidding algorithm.

Elementor-Based Landing Page

Heavy, slow page built with Elementor. 5–8 second load times, poor mobile experience for a campaign where 86.8% of traffic is mobile.

Real CPA Masked

Reported CPA of SGD 6.29 was false due to triple-counting. Real CPA is SGD 17.01 — nearly 3x what Google Ads was showing.

Performance Declining Since Feb 2026

Month-over-month conversion performance has been trending downward, compounded by the inflated data masking the real decline.

2 Actions Taken

All changes implemented to fix the identified issues across the landing page and conversion tracking.

Landing Page Rebuild

Rebuilt as Lightweight Raw HTML

Single file, no Elementor dependency. Stripped all page builder overhead for maximum speed and reliability.

Polished Hero Section with Modal Form

Mobile-First Design

Server-side form submission with transaction ID deduplication. One real conversion equals one counted conversion.

7 Inline CTAs Throughout the Page
